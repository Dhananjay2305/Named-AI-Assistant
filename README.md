# Named-AI-Assistant
We are building a named AI assistant for students and young professionals. Users can give it a name like Maya or Nova and talk to it daily for study planning, reminders, doubt solving, and career help. It works on WhatsApp and web. Unlike generic AI, it is personal, remembers user goals, and acts like a real assistant.

# NOVA AI – Named Personal AI Assistant 🤖✨

NOVA AI is a web-based AI assistant where users can create their own assistant by choosing a name (example: MAYA, NOVA, RAJU).  
The assistant helps users with study planning, reminders with reasons, note generation, and daily productivity.

## 🚀 Features
- Custom AI Assistant Name
- AI Chat (Study help + explanations)
- Smart Reminders (with reason)
- Study Planner (AI-generated timetable)
- Notes Vault

## 🛠 Tech Stack
- Frontend: React + TailwindCSS
- Backend: Node.js + Express
- Database: MongoDB Atlas
- AI: OpenAI API / Gemini API
